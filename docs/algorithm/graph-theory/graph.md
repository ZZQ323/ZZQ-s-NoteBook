# 图论
## 图基础


1. 路径，就是一堆相连的顶点及其边
   1. 从顶点v到w有路径，就称顶点v和m连通；
   2. 从顶点v到m有**路径**，且从顶点m到v有**路径**，则称这俩顶点时**强**连通的；
   3. 从顶点v到m有**单向路径**，则称这俩顶点时 **单向（半）** 连通的；
   4. **连通图是指任意两点都有路径**：
      1. 点和点之间能到就行，不需要有直接路径；
      2. 如果图中**任意俩顶点都连通，则该图为连通图**；
      3. 有向图中，若任意一对顶点都是强连通的，**称此图为强连通图**；
        ```mermaid
        graph LR
        subgraph Graph1[图1：强连通环]
            A1((A)) --> B1((B))
            B1 --> C1((C))
            C1 --> A1((A))
        end

        subgraph Graph2[图2：双向连通]
            A2((A)) --> B2((B))
            B2 --> A2
            C2((C)) --> B2
            B2 --> C2
        end
        Graph1 ~~~ Graph2
        ```
        
      4. 有向图中，任意两个顶点 u 和 v至少有一个方向存在路径，则成为**单向连通**
        ```mermaid
        graph LR
        A((A)) --> B((B))
        B --> C((C))
        A --> C
        ```
      5. 有向图只有在退化到无向的时候才能满足联通要求，则被称为**弱连通**；如果依旧不联通，则称之为**不联通**
        ```mermaid
        graph LR
        subgraph Graph1[图1：弱连通]
        A1((A)) --> B1((B))
        C1((C)) --> B1((B))
        end
        
        subgraph Graph2[图2：不联通]
        A2((A)) --> B2((B))
        C2((C))
        end
        Graph1 ~~~ Graph2
        ```
   5. （简单）**完全图是指任意两个点都直接相连**：
      1. 如果是一个完全图，它的顶点数是固定的；如果要研究他的性质，必然要从边上下手。
      2. 对于有向图，只考虑从该顶点出去的边或者只考虑射入该顶点的边；与之相关的出边有 $n-1$ 条总共有n个顶点，所以有向完全图有 $n*(n-1)$ 条边，而对于无向图，任意一条无向图的边应该是有向图对应位置边的一半，所以总边数为 $\frac{n*(n-1)}{2}$。
2. 分量，就是**无向图中的子图**，选几个点得到的东西
   1. 极大连通分量，要求连通子图包含其所有的边（**要求子图完整**）；
   2. 连通分量：要求子图**连通**，还要求该连通子图**极大**；
   3. 强连通分量：**有向图**中的极大强连通子图称为有向图的强连通分量
      1. 就是把多个强连通图放到放到一起，再挑出来
      2. 极大强连通分量：这里的极大和无向图完全一致
      3. 极小强连通分量：这里的极小和无向图完全一致
   4. 那么下面这张图的子图，除了一个点 `D`，除了 `A-B`、`A-B-G` …… 只有 `A-B-C-G`、`D-E-F` 是一个完整的子图，也就是极大；
    <div style="display: flex; justify-content: center;">
      
      ```mermaid
      graph LR
      A((A)) --- B((B))
      B --- C((C))
      B --- G((G))
      D((D)) --- E((E))
      D --- F((F))
      ```
   </div>
   <strong><div style="display: flex; justify-content: center;">图1. 想象:两个连通图放一起再挑出来</div></strong>
    
   5. 极小连通分量：极小是在保持连通的情况下使边数最少的子图，也就是“**是树的子图**”
      1. **树是包含所有顶点的极小连通图**；
      2. 生成树：连通图的生成树是包含图中全部顶点的一个极小连通子图
      3. 生成森林：在非连通图中，连通分量的生成树构成了非连通图的生成森林。
